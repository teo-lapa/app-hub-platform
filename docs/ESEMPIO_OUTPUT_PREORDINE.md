## 7. Utilizzo dei Dati Esportati

### Con JSON

```javascript
// Carica il file JSON
const data = require('./preordine_fornitori_2025-11-09T14-30-45.json');

// Trova prodotti di un fornitore specifico
const aligro = data.suppliers['ALIGRO Demaurex & Cie SA'];
console.log(`ALIGRO ha ${aligro.total_products} prodotti`);

// Trova tutti i fornitore con piÃ¹ di 100 prodotti
const bigSuppliers = Object.entries(data.suppliers)
  .filter(([name, data]) => data.total_products > 100)
  .map(([name, data]) => ({ name, products: data.total_products }));

console.log('Fornitori con >100 prodotti:', bigSuppliers);
```

### Con CSV in Excel

1. Apri il CSV in Excel
2. Seleziona tutti i dati (Ctrl+A)
3. **Dati** > **Raggruppa e filtra** > **Avanzate**
4. Applica filtro su colonna "Fornitore"
5. Crea tabella pivot:
   - Righe: Fornitore
   - Valori: Conteggio Nome Prodotto, Somma Prezzo Listino
6. Genera grafici di distribuzione
