### Problema: "Trovati 0 prodotti invece di 1,532"

**Controllo 1**: Verifica ID tag

```
Inventario > Configurazione > Product Tags
Cerca: "PRE-ORDINE"
Controlla ID (potrebbe non essere 314)
```

**Controllo 2**: Verifica che i prodotti abbiano il tag

```
Apri un prodotto manualmente
Tab: Inventario o Vendite
Cerca campo "Tags" o "Categorie"
Verifica presenza tag PRE-ORDINE
```
