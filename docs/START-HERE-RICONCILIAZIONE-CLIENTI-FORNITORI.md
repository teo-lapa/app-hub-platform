## WORKFLOW CORRETTO (DA SEGUIRE SEMPRE)

### Pagamenti Fornitori

**SBAGLIATO (Causa delle 603 anomalie):**
```
Ricevi bonifico bancario → Registra pagamento in Odoo → Fine
```

**CORRETTO:**
```
1. Ricevi fattura fornitore (email/cartaceo)
2. Registra fattura in Odoo (Acquisti > Fatture fornitori)
3. Approva fattura
4. Crea pagamento DA FATTURA (bottone "Registra pagamento")
5. Esegui bonifico bancario
6. Import estratto conto bancario
7. Riconcilia automaticamente (Odoo collega import a pagamento esistente)
```

**MAI:**
- Registrare pagamento senza fattura in Odoo
- Pagare fornitore senza fattura registrata
- Saltare step 2-4

### Incassi Clienti

**CORRETTO:**
```
1. Emetti fattura cliente in Odoo
2. Ricevi pagamento su conto bancario
3. Import estratto conto bancario
4. Riconcilia automaticamente (Odoo collega a fattura)
```

**Se pagamento senza fattura:**
- Verificare se è anticipo (registrare come anticipo cliente)
- Verificare se fattura non emessa (emettere fattura retroattiva)
- MAI lasciare pagamento "sospeso" senza collegamento
