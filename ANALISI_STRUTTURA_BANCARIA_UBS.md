## 8. GUIDA IMPLEMENTAZIONE IMPORTAZIONE UBS

### Step 1: Preparazione dati CSV

Il file CSV UBS deve contenere almeno:
- Data movimento
- Descrizione/Riferimento
- Importo
- Saldo (opzionale ma consigliato)

### Step 2: Creazione estratto conto

```javascript
const statement = await odoo.create('account.bank.statement', {
    name: 'UBS/2025/001',  // Nome estratto
    date: '2025-11-11',     // Data estratto
    journal_id: XX,         // ID giornale UBS (vedi sezione 1)
    balance_start: 0.0,     // Saldo iniziale
    balance_end_real: 0.0   // Saldo finale
});
```

### Step 3: Creazione righe movimento

```javascript
const line = await odoo.create('account.bank.statement.line', {
    statement_id: statement.id,
    date: '2025-11-11',
    payment_ref: 'Descrizione movimento',
    amount: 1000.00,        // Positivo per entrate, negativo per uscite
    partner_id: false,      // ID partner se identificato
    partner_name: 'Nome cliente',  // Nome se partner non trovato
});
```

### Step 4: Validazione estratto

Dopo l'importazione, verificare:
1. Saldo finale calcolato corrisponde al saldo reale
2. Tutte le righe sono state importate correttamente
3. Nessuna riga duplicata
